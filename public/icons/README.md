# ICC WebRadio - Fichiers d'icônes temporaires

Ce dossier contient des fichiers d'icônes temporaires pour le manifeste PWA. Des icônes réelles doivent être ajoutées ici :

- icon-192.png (192x192 pixels)
- icon-512.png (512x512 pixels) 
- icon-512-maskable.png (512x512 pixels, version maskable)

## Comment créer des icônes réelles :

Vous pouvez générer ces icônes en utilisant des outils en ligne comme :
- https://www.favicon-generator.org/
- https://maskable.app/
- Ou en utilisant un éditeur graphique comme Photoshop, GIMP ou Figma

Les icônes doivent être carrées et correspondre à l'image de marque d'ICC WebRadio.
